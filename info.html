Explaining HTML, HTML5, and CSS

By:James Kalvitz

What is HTML?

HTML stands for "Hyper Text Markup Language". HTML is the standard markup language and is typically used for creating Web pages. HTML consists of a series of elements that tell the browser how to display the content. The purpose of a web browser (Chrome, Edge, Firefox, Safari) is to read HTML documents and A browser does not display the HTML tags, but it will use them to determine how the document is displayed.

What is HTML5?

HTML5 is described as the newest version of HTML. HTML5 is the updated version compared to previous HTML versions and has introduced new elements and capabilities to the language on top of the previous versions. The term HTML5 is essentially a popular term that refers to a set of modern web technologies. Any modern site should use the HTML doctype as this will ensure that you are using the latest verion of HTML.

What is CSS?

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.
